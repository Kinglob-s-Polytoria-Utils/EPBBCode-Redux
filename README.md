# EPBBCode-Redux

If you've seen my repositories, you might've seen something called "`EPBBCode-polytora`".
`EPBBCode-polytora` is a great *concept*, but I executed it very poorly with low-quality code
which was unperformant and difficult to continue developing with.

This is why I, and another friend of mine are working on `EPBBCode-Redux`, meant to be faster,
easier to develop with, and cleaner.
